游戏介绍：根据提示依次挑战闯关，并通过答题来获取比分，在更短时间内，正确回答完题目的学员，可以获得更高分值。

难度等级：游戏有3个难度等级，每个难度等级的时间不同，每个等级有3小关。每个小关答对率达到80%，才能解锁下一关。

游戏时间：3小关的每题答题时间分别为20s,19s,18s，超时即算答错。

闯关失败说明：在闯关过程中，每个人有3次答错题目的机会，三次机会使用结束后，闯关失败。

体力值：每人每天有10次闯关的机会（体力值），用完后将无法进行闯关，体力值在第二天零时恢复。

得分规则：答对题的得分（100分/每题）+ 每题剩余时间(n)秒 x 10分，答错不得分。

排名规则：
    每一关只取最高历史得分，所有已通关的关卡得分总和计入排行榜。
    您的得分将被系统记录参与排名。同时您的成绩也将影响到您团队的成绩：

团队排名说明：
    以参与游戏的团队成员得分的加权平均分作为最终得分。
    团队中参与的人数越多，得分越高，则整个团队的得分越高。

团队得分计算公式：
    参与游戏用户平均分值 x 参与游戏的人数 / 团队总人数
    例如：
    团队A，有50人，其中25人玩了游戏，平均每人得分95分；
    团队B，有15人，其中5人玩了游戏，平均每人得分98分。
    团队A：95 x 25/50 = 47.5 分　　
    团队B：98 x 5/15 = 32.6分
